锂电排布优化系统 (Ternary Cell Pack Optimizer)
本项目是一款针对锂电池组（支持三元锂、磷酸铁锂等电芯）空间排布规划与干涉校验的辅助工具。系统支持 Web 部署及基于 Capacitor 的移动端打包，旨在通过对电池仓空间、电芯膨胀公差、BMS 刚体尺寸以及硅胶走线弯折体积的多维度约束计算，为工程人员提供排布参考。

🛠 核心系统特性
1. 全工艺细节物理建模 (Physical Process Modeling)
系统在参数设计上，不仅考虑了基础的物理长宽高，还引入了完整的锂电池组装工艺约束：

外壳厚度抵消：支持自定义电池仓外壳壁厚（如 shell-t 默认扣除 1mm 不锈钢厚度），自动计算出净空间。

电芯散热与膨胀公差：提供电芯发热或膨胀的散热间距 (cell-gap) 公差设置，降低实际组装中由于电芯胀气导致无法塞入金属外壳的风险。

装配辅料绝缘层：引入环氧板厚度 (epoxy-t)、连接片厚度 (tab-t) 以及铜铝复合片厚度 (composite-t) 计算，在空间规划中预留出绝缘和导电附件的必要体积。

硅胶主线走线硬约束：支持选择 10AWG 至 95mm² 的主流大电流线规，考虑高压粗线在弯折时极占空间的物理实际，系统可设定不同出线方向（顶部、侧面、端部）和数量来合理规避空间干涉。

2. 双重视图与干涉校验 (Multi-View Visualization)
📐 2D 平面绘制 (canvas)：动态绘制电芯俯视图，标示正负极（+/-）朝向，自动呈现排布走线，并支持显示串联焊接路径。

🧊 3D 空间仿真 (Three.js)：建立 WebGL 上下文，渲染电池仓透明外壳、堆叠电芯和 BMS 实体块，提供拖拽旋转交互，直观校验电池高度与顶盖空间的物理干涉。

📊 多型号并行对比：在对比面板（.cmp-table）中同步比对多组预设方案下的空间利用率与可行性，通过红绿状态徽章直观显示。

3. 寻优模拟与版本数据记录 (Optimization Simulation & Logging)
项目中包含独立的指标模拟与数据记录模块：

运行轮数追踪：内置 scripts/train.cjs 脚本。该脚本可模拟优化方案在系统中的运行迭代轮数，通过累加循环次数并在 skills.json 文件中持久化记录历史数据。

版本演进记录：用于在开发和调试过程中模拟算法的版本追踪与自增，为持续优化的工作流提供基础的数据记录支撑。

4. 实时工艺诊断 (Real-Time Diagnostics)
底部看板与交互模态框支持提供多维度的工艺参数评估：

空间利用率评估：计算并展示电芯占用体积与可用空间的占比。

走线长度估算：提供连接铜排总长度的计算估值，辅助评估回路内阻。

串联流程向导：提供独立的工艺步骤弹出框，依序渲染各节点的正负极连接流，用以辅助现场工人进行无差错组装。
